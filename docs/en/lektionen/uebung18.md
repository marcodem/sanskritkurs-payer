# Exercise 18

## Subhāṣita

Translate the subhāṣita at the beginning of the lesson:

नास्ति कामसमो व्याधिर्  
नास्ति मोहसमो रिपुः ।  
नास्ति क्रोधसमो वह्निर्  
नास्ति ज्ञानसमं सुखम् ॥

There is no disease like lust,  
There is no enemy like delusion,  
There is no fire like anger,  
There is no happiness like knowledge.

1. **antagata** (अन्तं गतः): gone to the end; in grammar: ending.
