# Lesson 31

## 31.1. The 7th Present Class (*rudhādi* = "rudh etc.")

**Formation:**
A nasal infix (**-na-** or **-n-**) is inserted before the final consonant of the deep-grade root:
*   **Strong Stem**: **-na-**
*   **Weak Stem**: **-n-**

The rules for combining the final consonants of the root with endings starting with consonants are the same as for the 2nd present class.

**Examples:**

### **yuj** 7U "to connect, to yoke"

| | | Parasmaipada | Ātmanepada |
| :--- | :--- | :--- | :--- |
| **Indicative** | | | |
| | **3. sg.** | *yunakti* (*yu-na-j + ti*) | *yuṅkte* (*yu + n + j + te*) |
| | **3. pl.** | *yuñjanti* (*yu + n + j + anti*) | *yuñjate* (*yu + n + j + ate*) |
| **Optative** | | | |
| | **3. sg.** | *yuñjyāt* (*yu + n + j + yā + t*) | *yuñjīta* (*yu + n + j + ī + ta*) |
| | **3. pl.** | *yuñjyur* (*yu + n + j + y + ur*) | *yuñjīran* (*yu + n + j + ī + ran*) |
| **Participle** | | *yuñjant-* (f. *yuñjatī*) | |

*Note: Nasals are replaced by -ṅ- before a guttural within a word.*

### **rudh** 7U "to stop, to hinder"

| | | Parasmaipada | Ātmanepada |
| :--- | :--- | :--- | :--- |
| **Indicative** | | | |
| | **3. sg.** | *ruṇaddhi* (*ru + na + dh + ti*) | *runddhe* (*ru + n + dh + te*) |
| | **3. pl.** | *rundhanti* (*ru + n + dh + anti*) | *rundhate* (*ru + n + dh + ate*) |
| **Optative** | | | |
| | **3. sg.** | *rundhyāt* (*ru + n + dh + yā + t*) | *rundhīta* (*ru + n + dh + ī + ta*) |
| | **3. pl.** | *rundhyur* (*ru + n + dh + y + ur*) | *rundhīran* (*ru + n + dh + ī + ran*) |
| **Participle** | | *rundhant-* (f. *rundhatī*) | |

In some roots, the **-n-** of the weak present stem has also penetrated into other non-present tenses, so these roots are listed with the infixed nasal.

Example: **bhañj** 7P "to break"
*   **Indicative Present**: *bhanakti* (3.sg.P), *bhañjanti* (3.pl.P).
*   **Future**: *bhaṅkṣyati* (from *bha + n + j + sya + ti*).
*   **Passive**: *bhajyate* (either from original root *bhaj* or from *\*bhñj-ya-te*).
*   **PPP**: *bhagna* (possibly from *\*bhñj + na*).

## 31.2. Vocabulary

*   **yuj** 7U (*yunakti*, Fut. *yokṣyati*, Pass. *yujyate*, Kaus. *yojayati*, PPP *yukta*, Inf. *yoktum*): to yoke, yoke up, fasten; **Ā** also: to exert oneself, to connect with, to concentrate on (Locative, *saptamī*).
    *   **yuga** n.: yoke, pair, world age (there are four ages: 1. *kṛta*, 2. *tretā*, 3. *dvāpara*, 4. *kali*).
    *   **yoga** m.: "yoking," effort, connection, Yoga.
    ![](/images/lekt3101.jpg)
*   **rudh** 7U (*ruṇaddhi*, Fut. *rotsyati*, Pass. *rudhyate*, Kaus. *rodhayati*, PPP *ruddha*, Inf. *roddhum*): to stop, bring to a standstill, hold back = enclose, cover.
*   **chid** 7U (*chinatti*, Fut. *chetsyati*, Pass. *chidyate*, Kaus. *chedayati*, PPP *chinna*, Inf. *chettum*): to cut off.
*   **bhañj** 7P (*bhanakti*, Fut. *bhaṅkṣyati*, Pass. *bhajyate*, PPP *bhagna*): to break (something).
*   **añj** 7P (*anakti*, Fut. *aṅkṣyati/añjiṣyati*, Pass. *ajyate*, Kaus. *añjayati*, PPP *akta*, Inf. *añjitum/aṅktum*): to anoint, smear.
*   **añj + vi** 7Ā (*vyaṅkte*, PPP *vyakta*): to smear apart = to make up (one's face), to distinguish oneself.
    *   **vyañjana** n.: means of distinction = makeup, spice, characteristic mark, consonant (that through which meanings are distinguished).
    ![](/images/lekt3102.jpg)
*   **bhid** 7U (*bhinatti*, Fut. *bhetsyati*, Pass. *bhidyate*, Kaus. *bhedayati*, PPP *bhinna*, Inf. *bhettum*): to split.
*   **bhuj** 7U (*bhunakti*, Fut. *bhokṣyati*, Pass. *bhujyate*, Kaus. *bhojayati*, PPP *bhukta*, Inf. *bhoktum*): to enjoy (e.g., food; "enjoy the earth" = to rule the earth).
    *   **bhoga** m.: enjoyment, food, pleasure, benefit, tax.
*   **bandh** 9P (*badhnāti*!, Fut. *bhantsyati*, Pass. *badhyate*, Kaus. *bandhayati*, PPP *baddha*, Inf. *banddhum*): to bind, tie up.
    *   **bandhana** n.: binding, fetter.
*   **jñā + pra** 9U (*prajānāti*): to recognize, understand.
    *   **prajñā** f.: wisdom, insight.
    ![](/images/lekt3103.jpg)
*   **bhū + sam** 1P (*sambhavati*): to arise, exist.
*   **śarīra** n.: body.
