# Lesson 57

## 57.1. The *s*-Aorist (Sigmatic Aorist #4)

The *s*-Aorist is formed from most **aniṭ** roots (those that do not take the connecting vowel *-i-*). It is also the standard Ātmanepada form for roots that take a non-sigmatic Aorist in the Parasmaipada.

**Formation**: **Augment + Root + s + Athematic Secondary Endings**

### Endings Table (with Suffix *-s-*)
| Person | Parasmaipada | Ātmanepada |
| :--- | :--- | :--- |
| | **Sing. / Plur.** | **Sing. / Plur.** |
| **1st** | **-sam / -sma** | **-si / -smahi** |
| **2nd** | **-sīs / -sta** | **-sthās / -dhvam** |
| **3rd** | **-sīt / -sur** | **-sta / -sata** |

**Note**: A sibilant (*s*) between two stops is lost (e.g., *a-kṣip + s + ta* → **akṣipta**). Traditional grammarians also drop the *-s-* after a short vowel in the 2nd/3rd person singular Ātmanepada (e.g., *a-kṛ + s + ta* → **akṛta**).

### Root Grades:
*   **Parasmaipada**: **Vṛddhi** (Long Grade).
    *   *nī* → **anaiṣīt** ("he led").
    *   *śru* → **ashrauṣīt** ("he heard").
    *   *kṛ* → **akārṣīt** ("he did").
*   **Ātmanepada**:
    *   Roots in *-i, -ī, -u, -ū*: **Guṇa** (High Grade). (e.g., *nī* → **aneṣṭa**).
    *   Roots in *-ṛ*: Replaced by **-īr** or **-ūr**.
    *   Roots in *-ā, -e, -o*: **Weak Grade**.
    *   Other roots: Usually unchanged (Weak Grade).

### Special Paradigms:

### Irregularities:
*   **dṛś** (to see) and **sṛj** (to emit) take the grade **-rā-** in the Parasmaipada: *adrākṣīt*, *asrāakṣīt*.
*   **pad** (4Ā) takes the root-aorist passive form in the 3rd per. sg.: **apādi** ("he went").
*   **budh** (1U) can take **aboddha** or **abodhi** in the 3rd per. sg. Ā.
