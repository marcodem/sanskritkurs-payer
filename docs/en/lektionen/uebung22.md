# Exercise 22

## Exercise: Absolutives (Gerunds)

Form and translate the Absolutive (Gerund) for the following verbs. Remember: simple roots take *-tvā*, while roots with prefixes take *-ya*.

1. **āp** -> *āptvā* (having obtained)
2. **pra-āp** -> *prāpya* (having reached/obtained)
3. **sam-ās** -> *samāsya* (having sat down together)
4. **āś** -> *āśitvā* (having eaten)
5. **sam-i** -> *samitya* (having come together)
6. **saṃ-skṛ** -> *saṃskṛtya* (having prepared/refined)
7. **kṛ** -> *kṛtvā* (having done)
8. **gam** -> *gatvā* (having gone)
9. **upa-gam** -> *upagatya* / *upagamya* (having approached)
10. **ji** -> *jitvā* (having conquered)
11. **vi-ji** -> *vijitya* (having conquered completely)
12. **dah** -> *dagdhvā* (having burned)
13. **apa-diś** -> *apadiśya* (having pointed out)
14. **nī** -> *nītvā* (having led)
15. **pac** -> *paktvā* (having cooked)
16. **budh** -> *buddhvā* (having known/awakened)
17. **sam-budh** -> *sambudhya* (having fully understood)
18. **bhū** -> *bhūtvā* (having been/become)
19. **pra-bhū** -> *prabhūya* (having prevailed)
20. **muk-tvā** -> *muktvā* (having released/liberated)
21. **vi-muc** -> *vimuçya* (having set free)

## Sentences & Compounds

Translate and analyze the compounds:

1. अन्नं पक्त्वा ब्राह्मणदास्यत्ति ॥१॥
   Having cooked the food, the Brahmin's maidservant (*brāhmaṇa-dāsī*) eats.
2. इष्टदेवतापूजां कृत्वेन्द्रादिदेवान्सद्ब्राह्मणाः स्तुवन्ति ॥२॥
   Having performed the worship of their chosen deity (*iṣṭa-devatā-pūjā*), the good Brahmins praise Indra and the other gods (*indrādi-devān*).
3. प्रस्थाय रामः सपुत्रः सद्guruश्रवणार्थेन ब्राह्मणग्रामं गच्छति ॥३॥
   Having set out, Rāma, together with his son (*sa-putraḥ*), goes to the Brahmin village for the purpose of listening to the good teacher.
4. अनिष्ट्वा नरो भगवद्भक्तिमात्रेणापि मोक्षमाप्नोति ॥४॥
   Even without having sacrificed (*aniṣṭvā*), a man reaches liberation (*mokṣam*) through mere devotion to the Lord (*bhagavad-bhakti-mātreṇa*).
