# Script Exercise 8

### Special Symbols

#### Avagraha
The elision of an initial **-a** (according to Sandhi rules like *-as + a-* » *-o '*) is marked by the **Avagraha** symbol (ऽ).

![](/images/schrift017.jpg)

Example:
*   *devo 'gniḥ* = देवो ऽग्निः
