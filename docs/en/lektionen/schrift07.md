# Script Exercise 7

### Ligatures (Samyuktākṣara) - Part 2: Special Case 'ra'

The consonant **ra** (र) behaves differently depending on whether it comes before or after another consonant.

#### 1. Preceding 'ra' (Repha)
If **ra** comes before another consonant, it is written as a small hook (ं) above the line, attached to the following consonant.
![](/images/schrift014.jpg)
Example: **r-ka** » र्क; **r-ma** » र्म.

#### 2. Following 'ra' (Rakra)
If **ra** follows another consonant, it is written as a diagonal stroke (्र) attached to the vertical bar of the preceding consonant.
![](/images/schrift015.jpg)
Example: **k-ra** » क्र; **p-ra** » प्र.

#### 3. Following 'ra' with rounded consonants
With consonants like **ṭa, ṭha, ḍa, ḍha**, it is written as a small wedge below the character.
![](/images/schrift016.jpg)
Example: **ṭ-ra** » ट्र.
