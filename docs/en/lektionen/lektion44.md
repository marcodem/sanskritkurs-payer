## 44.2. Internal Word Sandhi

When adding endings directly to consonant-ending stems, the following laws of word Sandhi apply:

### 44.2.1. Rules for *dh-*

1.  An **aspirate** is replaced by its non-aspirate before another aspirate:
    - *rundh* + *dhve* → **runddhve** (7U).
2.  **-h + dh-** → **-gdh-**:
    - *duh* + *dhve* → **dhugdhve** (2U).
3.  **-ṣ + dh-** → **-ḍḍh-** (specific to conjugation):
    - *dviṣ* + *dhve* → **dviḍḍhve** (2U).
4.  **-s + dh-** → **-dh-** (the *-s* drops out):
    - *ās* + *dhve* → **ādhve** (2Ā).

### 44.2.2. Rules for *s-*

1.  **-s + s-** → **-ts-** or (in Class 2) **-ss-**:
    - *vas* 1P (Fut.): *vas + sya + ti* → **vatsyati**.
    - *vas* 2Ā: *vas + se* → **vasse**.
2.  **-ṣ + s-** → **-kṣ-**:
    - *dviṣ* 2U: *dveṣ + si* → **dvekṣi**.

## 44.3. 2nd Person of Athematic Stems Without Suffix (Class 2)

### 44.3.1. Examples in Class 2 (*adādi*)

**dviṣ (2U)**
*   *laṭ*: *dvekṣi / dviṣṭha* (P), *dvikṣe / dviḍḍhve* (Ā).
*   *laṅ*: *adveṭ / adviṣṭa* (P), *adviṣthās / dviḍḍhvam* (Ā).
*   *vidhiliṅ*: *dviṣyās / dviṣyāta* (P), *dviṣīthās / dviṣīdhvam* (Ā).

**ās (2Ā)**: *āsse / ādhve* (*laṭ*); *āsthās / ādhvam* (*laṅ*).

**duh (2U)**
*   *laṭ*: *dhokṣi / dugdha* (P), *dhukṣe / dhugdhve* (Ā).
*   *laṅ*: *adhok / adugdha* (P), *adugdthās / adhugdhvam* (Ā).

**i (2P)**: *eṣi / itha* (*laṭ*); *ais / aita* (*laṅ*).

**han (2P)**: *haṃsi / hatha* (*laṭ*); *ahan / ahata* (*laṅ*).

**stu (2U)**: *stauṣi (stavīṣi) / stutha* (P).

**as (2P)**: **asi / stha** (*laṭ*); **āsīs / āsta** (*laṅ*); **syās / syāta** (*vidhiliṅ*).
*Note: The famous Mahāvākya "Tat tvam asi" (That thou art).*

**śās (2P)**: *śāssi / śiṣṭha* (*laṭ*); *aśās / aśiṣṭa* (*laṅ*).

**ad (2P)**: *atsi / attha* (*laṭ*); *ādas / ātta* (*laṅ*). Note thematic singular forms in Imperfect.

## 44.4. The Vocative (*āmantritam*)

The Vocative is used to address someone directly and stands outside the syntactic structure of the sentence. It is usually placed at the beginning:
*   *bāla kiṃ vadasi* = "Child, what are you saying?"

![](/images/lekt4401.jpg)

The plural and dual forms of the Vocative are identical to the Nominative. The singular forms will be covered in [Lesson 45](lektion45.md).

![](/images/lekt4402.jpg)

## 44.5. Vocabulary

*   **prati-** (Preverb): back, against, towards.
    *   **han + prati** 2P: to strike back.
    *   **vad + prati** 1P: to speak back, to answer.
    *   **khyā + prati + ā** 2P: to reject, refuse.
*   **yā** 2P (*yāti*, PPP *yāta*): to go, travel.
    *   **yāna** n.: going, path, vehicle.
    ![](/images/lekt4403.jpg)
*   **śī** 2Ā (*śete*, 3.pl.Ā *śerate*, PPP *śayita*): to lie down. Note that this root has High grade throughout the present stem.
    *   **śayana** n.: couch, bed.
    ![](/images/lekt4404.jpg)
*   **nand** 1P (*nandati*, PPP *nandita*): to rejoice in (Instr.).
    *   **nandin** m.: joyful; name of Shiva's bull (*vahana*).
    ![](/images/lekt4405.jpg)
*   **nand + abhi** 1P: to rejoice in (Acc.), to welcome.
*   **yam** 1P (*yacchati*, PPP *yata*): to hold, sustain; to give, grant; to curb, restrain.
    *   **yam + ā** 1U: to extend, stretch out (*āyata*).
        ![](/images/lekt4406.jpg)
    *   **yam + pra** 1P: to offer, hand over.
    *   **yam + sam** 1P: to bind together, curb.
*   **yat** 1Ā (*yatate*, PPP *yatta*): to strive for (Loc., Dat., Acc.).
    *   **yatna** m.: effort, exertion.
    ![](/images/lekt4407.jpg)
*   **rabh** 1Ā (*rabhate*, PPP *rabdha*): to seize.
    *   **rabh + ā** 1Ā: to begin, undertake.
*   **pravrajyā** f.: "going forth" from home to homelessness (Buddhist novice ceremony).
    ![](/images/lekt4408.jpg)
