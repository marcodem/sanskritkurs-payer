# Lesson 59

## 59.1. Secondary and Imperative Dual Endings (*dvivacanam*)

| Person | Parasmaipada (Sec. / Imp.) | Ātmanepada (Sec. / Imp.) |
| :--- | :--- | :--- |
| **1st** | **-va / -āva** | **-vahi / -āvahai** |
| **2nd** | **-tam** | **-īthām / -āthām** |
| **3rd** | **-tām** | **-ītām / -ātām** |

## 59.2. Dual of Thematic Present Classes (Opt., Impf., Imp.)

Before *-va* and *-vahi*, the thematic vowel *-a-* is lengthened to **-ā-**.

### Example: bhū (1P, "to be")
*   **Optative**: *bhaveva* (1st), *bhavetam* (2nd), *bhavetām* (3rd).
*   **Imperfect**: *abhavāva* (1st), *abhavatam* (2nd), *abhavatām* (3rd).
*   **Imperative**: *bhavāva* (1st), *bhavatam* (2nd), *bhavatām* (3rd).

### Example: cur (10U, "to steal")
*   **Optative**: *corayeva / corayevahi*.
*   **Imperfect**: *achorayāva / achorayāvahi*.
*   **Imperative**: *corayāva / corayāvahai*.

## 59.3. Dual of Athematic Classes

*   **Class 2 (as "to be")**: Opt. *syāva*, Impf. *āsva*, Imp. *asāva / stam*.
*   **Class 2 (han "to slay")**: Opt. *hanyāva*, Impf. *ahanva*, Imp. *hanāva / hatam*.
*   **Class 3 (dhā "to set")**: Opt. *dadhyāva*, Impf. *adadhva*, Imp. *dadhāva / dhattām*.
*   **Class 8 (kṛ "to do")**: Opt. *kuryāva*, Impf. *akurva*, Imp. *karavāva / kurutām*.

## 59.4. Dual of Aorists

*   **Root Aorist (pā)**: *apāva* (1st), *apātam* (2nd), *apātām* (3rd).
*   **a-Aorist (sic)**: *asicāva / asicāvahi* (1st).
*   **Reduplicated (shri)**: *ashishriyāva / ashishriyāvahi* (1st).
*   **s-Aorist (nī)**: *anaiṣva / aneṣvahi* (1st).
*   **iṣ-Aorist (pū)**: *apāviṣa / apaviṣvahi* (1st).

## 59.5. The *siṣ*-Aorist (Sigmatic Aorist #6)

This form is used in the **Parasmaipada only** for roots ending in *-ā, -e, -o, -ai*, and a few others (e.g., *nam, yam, ram*). In the Ātmanepada, these roots use the *s*-Aorist.

**Formation**: **Augment + Root + siṣ + Athematic Secondary Endings**

### Example: yā (2P, "to go")
| Person | Singular | Dual | Plural |
| :--- | :--- | :--- | :--- |
| **1st** | *ayāsiṣam* | *ayāsiṣva* | *ayāsiṣma* |
| **2nd** | *ayāsiḥ* | *ayāsiṣṭam* | *ayāsiṣṭa* |
| **3rd** | *ayāsiṭ* | *ayāsiṣṭām* | *ayāsiṣur* |

## 59.6. The *sa*-Aorist (Sigmatic Aorist #7)

The thematic sigmatic Aorist, used by roots ending in *-ś, -ṣ, -h* with a penultimate short vowel (*i, u, ṛ*).

**Formation**: **Augment + Weak Root + sa + Thematic Secondary Endings**

### Example: dish (6U, "to show")
*   P: *adikṣam* (1st sg.), *adikṣāva* (1st du.), *adikṣat* (3rd sg.).
*   Ā: *adikṣi* (1st sg.), *adikṣāvahi* (1st du.), *adikṣata* (3rd sg.).
