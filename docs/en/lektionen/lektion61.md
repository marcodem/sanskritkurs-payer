# Lesson 61

## 61.1. Declension of Root Nouns in *-ā, -ī, -ū*

### 61.1.1. Root Nouns in *-ā* (at the end of *Tatpuruṣas*)
Before vocalic endings, the stem vowel is dropped in the weak cases.
*   **vishvapā** ("protecting the universe"): Nom. *vishvapās*, Acc. *vishvapām*, Dat. *vishvape*, Gen./Abl. *vishvapas*.

### 61.1.2. Feminine Root Nouns in *-ī*
*   **dhī** ("thought"): Nom. *dhīs*, Acc. *dhiyam*, Instr. *dhiyā*; Dat. *dhiye / dhiyai*.
*   **strī** ("woman"): Nom. *strī*, Acc. *striyam / strīm*, Instr. *striyā*, Gen. pl. *strīṇām*. ![](/images/lekt6102.jpg)

### 61.1.3. Root Nouns in *-ī* (at the end of *Tatpuruṣas*)
*   **shuddhadhī** ("pure-minded"): Nom. *shuddhadhīs*, Acc. *shuddhadhyam*.

### 61.1.4. Feminine Root Nouns in *-ū*
*   **bhū** ("earth"): Nom. *bhūs*, Acc. *bhuvam*, Instr. *bhuvā*.

### 61.1.5. Root Nouns in *-ū* (at the end of *Tatpuruṣas*)
*   **khalapū** ("sweeping the barn"): Nom. *khalapūs*, Acc. *khalapvam*. ![](/images/lekt6106.jpg)

### 61.1.6. Polysyllabic Feminine Nouns in *-ū*
*   **vadhū** ("bride, young woman"): Nom. *vadhūs*, Acc. *vadhūm*, Instr. *vadhvā*. ![](/images/lekt6107.jpg)

## 61.2. The Periphrastic Future (*luṭ*)

The Periphrastic Future (*luṭ*) is traditionally used for the distant future ("not today"). It is formed by the agent noun in **-tṛ** + the present of **as** (to be).

**Example: dā (to give)**
| Person | Singular | Dual | Plural |
| :--- | :--- | :--- | :--- |
| **1st** | *dātāsmi* | *dātāsvas* | *dātāsmas* |
| **2nd** | *dātāsi* | *dātāsthas* | *dātāstha* |
| **3rd** | **dātā** | **dātārau** | **dātāras** |

## 61.3. The Intensive (Frequentative) (*carkarītam*)

The Intensive denotes frequent, intensive, or (with verbs of motion) crooked/zig-zag action.

**Formation:**
1.  **Ātmanepada**: Redupl. Root + **-ya-** (e.g., *bobhūyate* "it happens repeatedly").
2.  **Parasmaipada**: Athematic conjugation + Strong Reduplication (e.g., *bobhavīti*).

Examples:
*   *bhū* → **bobhūyate / bobhavīti**
*   *vraj* (go) → **vavrajyate** ("moves in a zig-zag"). ![](/images/lekt6104.jpg)

## 61.4. Denominatives (*nāmadhātu*)

Verbs derived from nouns.
*   **Plain (Class 1)**: Acting like X (e.g., *putrati* "acts like a son").
*   **With -ya (P)**: Wishing for X / Treating as X (e.g., *putrīyati* "wishes for a son").
    *   **namasyati**: "he pays homage" (from *namas*). ![](/images/lekt6103.jpg)
    *   **tapasyati**: "he performs austerities." ![](/images/lekt6105.jpg)
*   **With -ya (Ā)**: Becoming/Acting as X (e.g., *kumārāyate* "acts like a girl").
*   **With -aya**: Various meanings (e.g., *muṇḍayati* "he shaves the head"). ![](/images/lekt6113.jpg)

## 61.5. The Benedictive (*āshīrliṅ*)

Used for blessings or expressions of hope.
*   *bhū* → **bhūyāt** ("May it be!"). ![](/images/lekt6114.jpg)

## 61.6. The Conditional (*lṛṅ*)

Used for counterfactual conditions ("If X had been, then Y would have been").
**Formation**: **Augment + Future Stem + Secondary Endings**.
*   *abhaviṣyat*: "It would have been."

## 61.7. Declension of *-ai, -o, -au* Nouns
*   **go** (m./f., "ox/cow"): Nom. *gaus*, Acc. *gām*, Gen. pl. *gavām*. ![](/images/lekt6115.jpg)

## 61.8. Remaining Pronouns
*   **Dual Personal**: **āvām** (we two), **yuvām** (you two). ![](/images/lekt6117.jpg)
*   **adas** ("that yonder"): Nom. *asau* (M/F), *adas* (N).
