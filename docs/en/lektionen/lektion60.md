# Lesson 60

## 60.1. Perfect Dual Endings (*dvivacanam*)

| Person | Parasmaipada | Ātmanepada |
| :--- | :--- | :--- |
| **1st** | **-va** | **-vahe** |
| **2nd** | **-athur** | **-āthe** |
| **3rd** | **-atur** | **-āte** |

## 60.2. Dual of the Reduplicated Perfect (*dvittvalṭ*)

The Dual forms generally follow the **weak stem** (except where specifically noted for Parasmaipada singular).

### Examples:
*   **No Gradation (bandh)**: *babandhiva* (1st P), *babandhatur* (3rd P).
*   **Strong/Weak Grade (bhid)**: *bibhidiva* (1st P), *bibhidāte* (3rd Ā).
*   **Roots in -ā / -ai (dā)**: *dadiva / dadivahe* (1st). Note that *dadāthe* and *dadāte* are identical to the Present Ind. forms!
*   **Consonant-a-Consonant (pac)**: *peciva / pecivahe* (1st).

## 60.3. Dual of the Periphrastic Perfect (*anuprayogaliṭ*)

*   **gaṇ (10P)**: *gaṇayāñ cakṛva / babhūviva*.
*   **ās (2Ā)**: *āsāṃ cakṛvahe / babhūviva*.

## 60.4. Perfect Parasmaipada Participle

**Formation**: **Weak Perfect Stem + vāṃs**
(Roots with a monosyllabic weak stem often insert the binding vowel *-i-*).

**Declension**:
*   Strong Stem: **-vāṃs**
*   Weak Stem (before consonant): **-vat**
*   Weak Stem (before vowel): **-uṣ** (Note: the binding vowel *-i-* is dropped before *-uṣ*).

**Feminine**: Weak stem (vocalic) + **-ī** (e.g., **viduṣī** "a wise woman").

## 60.5. Stems in *-añc* (*-ac*)

These stems are compounds of prefixes and the root noun of *añc / ac* (to move).

### 60.5.1. Three-Stem Type
*   Strong: **-añc**
*   Weak (cons.): **-ac**
*   Weak (vocalic): **-īc** (or **-īc / -ūc** after semivowels).
    *   *pratyañc* → *pratīc* (West)
    *   *udañc* → *udīc* (North)

### 60.5.2. Two-Stem Type
*   Strong: **-añc**
*   Weak: **-ac**
    *   *prāñc* (East)
    *   *avāñc* (Downwards)

**Cardinal Directions**:
*   North: **udañc**
*   South: **dakṣiṇa**
*   East: **prāñc**
*   West: **pratyañc**

## 60.6. The Desiderative (*san*)

The Desiderative expresses the **wish** to do something or being **about to** do something.

**Formation**: **Reduplicated Root + (i) + ṣa**
(Most roots use *-sa*, but *seṭ* roots and Class 10/Causatives use *-iṣa*).

### Rules for the Stem:
1.  **Weak Grade**: Usually preferred (e.g., *jīv* → **jijīviṣa**).
    *   Final *-i/-u* becomes *-ī/-ū* (e.g., *ji* → **jigīṣa** "wishing to conquer").
    *   Final *-ā* roots often contract (e.g., *dā* → **ditsa**, *dhā* → **dhitsa**).
2.  **Reduplication**: Usually uses the vowel **-i-** (e.g., *pā* → **pipāsa** "thirsty").
3.  **Vowel-Initial Roots**: Use "Attic Reduplication" (e.g., *īkṣ* → **īcikṣiṣa**).

### Conjugation:
Desiderative stems are conjugated like thematic verbs (Class 1).
*   **yiyakṣati**: "He wishes to sacrifice (for another)."
*   **yiyakṣate**: "He wishes to sacrifice (for himself)."
