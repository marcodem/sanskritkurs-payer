# Lesson 30

## 30.1. The 9th Present Class (*kryādi* = "krī etc.")

**Formation:**
*   **Strong Stem**: deep-grade root + **-nā-**
*   **Weak Stem**:
    - Before consonants: deep-grade root + **-nī-**
    - Before vowels: deep-grade root + **-n-**

Example:
*   **krī** 9U "to buy"

| | Singular | Plural |
| :--- | :--- | :--- |
| **3rd Pers. Parasmaipada** | *krīṇāti* (*krī + nā + ti*) | *krīṇanti* (*krī + n + anti*) |
| **3rd Pers. Ātmanepada** | *krīṇīte* (*krī + nī + te*) | *krīṇate* (*krī + n + ate*) |

**Note the formation for jñā 9U "to know":**
*   **Strong**: *jānāti* (*jā-nā-ti*)
*   **Weak**: *jānīte / jānanti* (*jā-nī / jā-n*)
(The base *jā-* stems either from an ablaut series or dissimilation from *\*jñā-nā-ti*).

Some roots with long vowels shorten them before the 9th class suffix:
Example:
*   **pū** 9U "to purify" → *punāti, punante*.

**Present Participle Parasmaipada:**
*   *krīṇant* (*krī + n + ant*); Fem.: *krīṇatī* (*krī + n + at + ī*).

## 30.2. Use of the Optative (*liṅ*)

The Optative is used to denote:
1.  **A wish, a request, or a mild command** (overlaps with the Imperative).
    > *dāso grāmam āgacchet* = "The servant should come to the village."
2.  **Possibility, probability, or assumption.**
    > *grāmācced gacched guruṃ na śṛṇuyāt* = "If he were to go from the village, he would not hear the master."
3.  **Relative clauses** with an Optative sometimes mean "If someone...":
    > *yo nṛtaṃ vadet sa narakaṃ patet* = "If someone were to speak untruth, he would fall into hell."

## 30.3. Secondary Endings (3rd Person Sg. and Pl.)

The Optative (*liṅ*), Imperfect (*laṅ*), Aorist (*luṅ*), Precative, and Conditional use the **secondary endings**:

| | 3rd Person Singular | 3rd Person Plural |
| :--- | :--- | :--- |
| **Parasmaipada** | *-t* | *-n* (Athematic: *-an* or *-ur*; Optative: *-ur*) |
| **Ātmanepada** | *-ta* | *-nta* (Athematic: *-ata*; Optative: *-ran*) |

## 30.4. Formation of the Optative Present (*liṅ*)

### 30.4.1. Thematic Present Classes (1, 4, 6, 10)

*   Before consonant-initial endings: **Pr. stem + -i-** (merges with *-a-* to **-e-**) **+ secondary ending**.
*   Before vowel-initial endings: **Pr. stem + -i-** (→ **-e-**) **+ -y- + secondary ending**.

Examples (3rd pers.):
*   **1. bhū**: *bhavet* (*bhava + i + t*) / *bhaveyur* (*bhava + i + y + ur*). Ā: *bhaveta / bhaveran*.
*   **10. cur / Kaus**: *corayet / corayeyur*. Ā: *corayeta / corayeran*.

### 30.4.2. Athematic Present Classes (2, 5, 8, 9)

*   **Parasmaipada**: **Weak Pr. stem + -yā-** (before *-ur*: **-y-**) **+ secondary ending**.
*   **Ātmanepada**: **Weak Pr. stem + -ī- + secondary ending**.

Examples (3rd pers.):
*   **2. dviṣ**: *dviṣyāt / dviṣyur*. Ā: *dviṣīta / dviṣīran*.
*   **5. su**: *sunuyāt / sunuyur*. Ā: *sunvīta / sunvīran*.
*   **8. kṛ**: *kuryāt / kuryur*. Ā: *kurvīta / kurvīran*.
*   **9. krī**: *krīṇīyāt / krīṇīyur*. Ā: *krīṇīta / krīṇīran*.

## 30.5. Sandhi of Final *-r*

Except for *-ar*, final **-r** follows the same sandhi rules as final **-s**.
**-ar** before voiced sounds remains **-ar**; before **r-**, the first **-r** drops and the **-a-** lengthens to **-ā-**.

Examples:
*   *bhaveyur + ca* → *bhaveyuś ca*
*   *punar + agniḥ* → *punar agniḥ*
*   *punar + roditi* → *punā roditi*
