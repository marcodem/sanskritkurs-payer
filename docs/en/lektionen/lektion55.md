# Lesson 55

## 55.1. The *a*-Aorist (Thematic Aorist)

The *a*-Aorist is formed similarly to the Imperfect (*laṅ*) of the 6th present class (*tudādi*):

**Augment + weak-grade root + -a- + secondary endings**

The *-a-* becomes *-ā-* before *-v-* and *-m-* of the endings, just like in the thematic present and imperfect. Note that nasal infixes used in the present stem (like those in Class 7 or Class 6) are **not** present here.

The Ātmanepada is rare; for roots that take both (*Ubhayapada*), it is typically formed according to other Aorist types.

**Example: sic (6U, "to sprinkle")**
| Person | Parasmaipada | Ātmanepada |
| :--- | :--- | :--- |
| | **Sing. / Plur.** | **Sing. / Plur.** |
| **1st** | *asicam / asicāma* | *asice / asicāmahai* |
| **2nd** | *asicas / asicata* | *asicathās / asicadhvam* |
| **3rd** | *asicat / asican* | *asicata / asicanta* |

### Irregular Formations:
*   **as** (4P, "to throw"): *āstham, āsthat*.
*   **khyā + ā** (2P, "to tell"): *akhyam, akhyat*.
*   **dṛś** ("to see"): *adarśam, adarśat*. (Note the Guṇa instead of weak grade).
*   **śās** (2P, "to command"): **aśiṣat** (regular weak grade of *śās* is *śiṣ*).
