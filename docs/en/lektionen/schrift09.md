# Script Exercise 9

### Word Separation

Within a sentence (or half-verse), words are only written separately in the script if a word ends with:
- a vowel
- Anusvāra (ṃ)
- Visarga (ḥ)
and the following word begins with a consonant. Also in cases where a hiatus (vowel gap) exists after Sandhi rules.

### Punctuation

- A single vertical bar (**|**, *ardhadaṇḍa*) marks the end of a sentence in prose or a half-verse.
- A double vertical bar (**||**, *daṇḍa*) marks the end of a verse or a paragraph.
- Verse numbers are placed between double bars, e.g., **||१||**.

### Abbreviations

The abbreviation mark is a small circle (**°**).
Example: **पा°** = *pā(ṇinīye)* = "In Pāṇini's work".
