# Lesson 46

## 46.1. Formation of 2nd Person Verbal Forms in the Perfect (*liṭ*)

### 46.1.1. Endings

| Type | Parasmaipada | Ātmanepada |
| :--- | :--- | :--- |
| | **Sing. / Plur.** | **Sing. / Plur.** |
| **Perfect** | *-tha / -a* | *-se / -dhve* |

*Note: In the Parasmaipada plural, the ending (-a) is identical to the 1st and 3rd singular endings, though the stem form may differ.*

### 46.1.2. Sound Changes for *-dhve*
The ending **-dhve** is replaced by **-ḍhve** if a root-final **-u** or **-ṛ** immediately precedes it. This change is optional after the union-vowel **-i-** if that vowel is preceded by a semivowel or *h*.

### 46.1.3. Perfect Type I: No Stem Variation
(Root structure: C-long V-C, C-V-C-C, a-C(C), ā-C)
*   **bandh** 9P → *babandhitha / babandha*.
*   **jīv** 1P → *jijīvitha / jijīva* (P); *&lt;jijīviṣe / jijīviḍhve&gt;* (Ā).
*   **as** 2P → *āsitha / āsa*.

### 46.1.4. Perfect Type II: Guṇa in Strong, Zero-grade in Weak Stems
(Root structure: (C-)i/u/ṛ/ḷ-C)
*   **bhīd** 7U → *bibheditha / bibhida* (P); *bibhidiṣe / bibhididhve* (Ā).
*   **muh** 4P → *mumohitha (mumoḍha) / mumuha*.

### 46.1.5. Perfect Type III: Vowel-Ending Roots

#### 46.1.5.1. Type IIIa: Zero-grade in Weak Stems
*   **i** 2P → *iyetha / īya*.
*   **nī** 2U → *ninayitha (ninetha) / ninya* (P); *ninyiṣe / ninyiḍhve* (Ā).
*   **stu** 2U → *tuṣṭotha / tuṣṭuva* (P); *tuṣṭuṣe / tuṣṭuḍhve* (Ā).
*   **kṛ** 8U → *cakartha / cakra* (P); *cakṛṣe / cakṛḍhve* (Ā).

#### 46.1.5.2. Type IIIb: Guṇa in Weak Stems
*   **pṝ** 3P → *paparith / papara*.

### 46.1.6. Perfect Type IV: Roots in *-ā* or *-ai*
*   **dā** 3U → *dadātha (daditha) / dada* (P); *dadiṣe / dadidhve* (Ā).
*   **gai** 1P → *jagātha (jagitha) / jaga*.

### 46.1.7. Perfect Type V: (C-)a-C Roots

#### 46.1.7.1. Type Va: Weak Stem Reduction
*   **gam** 1P → *jagamitha (jagantha) / jagma*.
*   **han** 2P → *jaghanitha (jagantha) / jagna*.
*   **jan** 4Ā → *jajñiṣe / jajñidhve*.
*   **yaj** 1U → *iyajitha (iyaṣṭha) / īja*.
*   **vac** 1P → *uvacitha (uvaktha) / ūca*.
*   **vah** 1U → *uvahitha (uvodḍha) / ūha*.

#### 46.1.7.2. Type Vb: Weak Stem with *-e-* (No Reduplication)
*   **pac** 1U → *peciitha / peca* (P); *peciṣe / pecidhve* (Ā).

#### 46.1.7.3. Type Vc: Guṇa in Weak Stems
*   **kram** 1U → *cakramitha / cakrama*.

### 46.1.8. Special Perfects
*   **vid** 2P (Present Perfect): *vettha / vida* ("thou knowest / you know").
*   **ah**: *āttha* ("thou sayest").
*   **bhū** 1P: *babhūvitha / babhūva*.
*   **ji** 1P: *jigetha / jigya*.

### 46.1.9. Periphrastic Perfect (*anuprayoga-liṭ*)
Formed by derived stems (Causatives, etc.) and roots beginning with long vowels.
*   **bandh** (Caus.) → *bandhayāṃ cakartha / cakṛṣe* (etc.).
