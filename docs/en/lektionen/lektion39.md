# Lesson 39

## 39.1. Personal Endings of the First Person

The endings for the 1st person (called "third person" / *tṛtīyaḥ puruṣaḥ* in traditional Sanskrit grammar) are:

| | Parasmaipada | Ātmanepada |
| :--- | :--- | :--- |
| | **Singular** | **Plural** | **Singular** | **Plural** |
| **Primary (Thematic)** | *-mi* | *-mas* | *-i* (see note) | *-mahe* |
| **Primary (Athematic)** | *-mi* | *-mas* | *-e* | *-mahe* |
| **Secondary (Them.)** | *-m* | *-ma* | *-i* | *-mahi* |
| **Secondary (Athem.)** | *-am* | *-ma* | *-i* | *-mahi* |
| **Optative** | *-am* | *-ma* | *-a* | *-mahi* |
| **Perfect** | *-a* | *-ma* | *-e* | *-mahe* |

*Note: For thematic Ātmanepada primary endings, the thematic vowel -a- merges with -i to form -e (e.g., bhava + i = bhave).*

## 39.2. Formation for Thematic Stems (Classes 1, 4, 6, 10)

**Rule:**
Before **-mi, -mas, -ma, -mahe, -mahi**, the thematic vowel **-a-** is replaced by **-ā-**.

### Examples:
*   **bhū** 1P: *bhavāmi* (1.sg.P), *bhavāmas* (1.pl.P); *abhave* (1.sg.Ā), *bhavāmahe* (1.pl.Ā).
*   **nṛt** 4P: *nṛtyāmi*, *nṛtyāmas*.
*   **viś** 6P: *viśāmi*, *viśāmas*.
*   **cur** 10U: *corayāmi*, *coraye*.
*   **Passive**: *īkṣye*, *īkṣyāmahe*.

## 39.3. First Person of the Future (*lṛṭ*)

The Future is thematic, so the same rule applies (*-a-* becomes *-ā-* before *m-* endings).
*   **dā** 3U: *dāsyāmi*, *dāsyāmas*, *dāsye*, *dāsyāmahe*.
*   **bhū** 1P: *bhaviṣyāmi*, *bhaviṣyāmas*.

## 39.4. Formation for Athematic Stems

In internal sandhi (within word formation), consonants generally remain unchanged before endings starting with a vowel, semivowel, or nasal.

### 39.4.1. Second Class (*adādi*)
*   **dviṣ** 2U: *dveṣmi*, *dviṣmas*; *dviṣe*, *dviṣmahe*.
*   **as** 2P: *asmi* (1.sg.P), *smas* (1.pl.P).
    ![](/images/lekt3901.jpg)

### 39.4.2. Third Class (*juhotyādi*)
*   **hu** 3P: *juhomi*, *juhumas*.
*   **dhā** 3U: *dadhāmi*, *dadhmhas*; *dadhe*, *dadhmhahe*.

### 39.4.3. Fifth Class (*svādi*) and Eighth Class (*tanādi*)
Roots ending in a vowel (Class 5) or the *-u* suffix (Class 8) can optionally drop the *-u-* before endings starting with *m-*.
*   **su** 5U: *sunomi*, *sunumas* (or *sunmas*).
*   **tan** 8U: *tanomi*, *tanumas* (or *tanmas*).
*   **kṛ** 8U: *karomi*, *kurmas* (always dropped).

### 39.4.4. Seventh Class (*rudhādi*)
*   **yuj** 7U: *yunajmi*, *yuñjmas*; *yuñje*, *yuñjmahe*.

### 39.4.5. Ninth Class (*kryādi*)
*   **krī** 9U: *krīṇāmi*, *krīṇīmas*; *krīṇe*, *krīṇīmahe*.
