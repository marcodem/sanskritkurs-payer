# Script Exercise 6

### Ligatures (Samyuktākṣara) - Part 1

When two or more consonants appear without a vowel between them, they are combined into a **ligature** (conjunct consonant).

### Rule 1: Consonants with a Vertical Bar

If the first consonant has a vertical vertical bar (like **kha, ga, gha, ca, ja, ta, tha, da, na, pa, ba, bha, ma, ya, la, va, śa, ṣa, sa**), the bar is removed and the remainder is joined to the following consonant.

![](/images/schrift013.jpg)

Examples:
- **stha** (s + tha): स्थ
- **nya** (n + ya): न्य
- **vya** (v + ya): व्य
