# Exercise 60

A) Learn the irregular desiderative formations for the verbs learned so far in Kielhorn's *Grammar*, § 451.

B) Identify and translate the following forms without aids:

2.  अहिंसीः - हिंस् 7P 2.sg.Aor(5).P "you have injured"

5.  अक्षथाः - क्षन् 8U 2.sg.Aor(5/1).Ā "you injured (for yourself)"

7.  अचिक्षंसेथाम् - क्षम् 1Ā Desid. 2.du.Ā "you two wished to endure"
8.  अस्नाः - स्ना 2P 2.sg.Impf.P "you bathed"
