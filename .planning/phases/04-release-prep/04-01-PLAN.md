# Phase 4 Plan: Build-Automatisierung & Lizenzen

This final phase ensures the project is ship-ready, verified for content integrity, and legally compliant.

## Path to Success

### Wave 1: Global Content Audit & Cleanup
- [ ] Task 1: **Run Integrity Sweep**. Create a script to scan all 60+ lessons for conversion leftovers (HTML tags, non-UTF8 chars, broken links).
- [ ] Task 2: **Fix Identified Issues**. Atomically fix common artifacts using bulk replace or script updates.

### Wave 2: License & Asset Finalization
- [ ] Task 3: **License Audit**. Cross-reference `docs/licenses.md` with actual image usage in chapters. Ensure every image has a corresponding entry.
- [ ] Task 4: **Project Information Update**. Finalize `docs/impressum.md` and `docs/index.md` with project history and latest status.

### Wave 3: Production Build & Preservation
- [ ] Task 5: **Verify Production Build**. Run `npm run docs:build` and inspect the `dist` folder.
- [ ] Task 6: **Reference Protection**. Confirm the `/sanskritkurs` original folder is intact and correctly linked for archival purposes.
