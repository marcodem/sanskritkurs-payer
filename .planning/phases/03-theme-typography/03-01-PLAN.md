# Phase 3 Plan: "Warmes" Theme & Typografie

This plan focuses on elevating the visual identity of the Sanskrit course to match the premium "Academic Scholarly" look established in the PoC.

## Path to Success

### Wave 1: Typography & Sanskrit Refinement
- [ ] Task 1: **Enable Devanāgarī Ligatures**. Update `custom.css` to activate advanced OpenType features for "Source Serif 4".
- [ ] Task 2: **Refine Global Spacing**. Adjust line heights and font-weights to improve legibility of dense Sanskrit text and IAST diacritics.

### Wave 2: Scholarly Components
- [ ] Task 3: **Implement "Scholarly Tables"**. Update `custom.css` with a custom styling for tables that avoids harsh borders and uses subtle parchment-based background shifts.
- [ ] Task 4: **Minimalist Content Start**. Implement CSS to hide redundant page headers and metadata areas, ensuring the technical Nav Bar stays for functionality.

## Tasks

Update `docs/.vitepress/theme/custom.css` to include the following for all body and heading elements:

html {
  font-variant-ligatures: common-ligatures, discretionary-ligatures;
  font-feature-settings: "kern", "liga", "clig", "dlig";
  text-rendering: optimizeLegibility;
}

Ensure "Source Serif 4" is the primary font for text blocks containing Sanskrit.
</action>

</task>

## Verification Criteria

- [ ] Devanāgarī text looks "connected" and professional (ligatures active).
- [ ] Tables are easy to read and look "academic" (no grid-box look).
- [ ] Each page starts immediately with the relevant lesson content.
- [ ] The Site Header (Search/Home) is still fully accessible.

## must_haves

- Design must feel "Parchment" based (warm).
- Typography must prioritize IAST and Devanāgarī clarity.
